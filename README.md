# SALE APP

Приложение позволяет публиковать объявления о продаже вещей, а также просматривать и комментировать объявления других пользователей. Для большей части функционала требуется аутентификация, доступен личный кабинет с функцией обновления информации о пользователе.

## Технологии
**Backend Framework**: Spring Boot 2.7, Java 11, Maven

**Database**: PostgreSQL, H2 Database, Spring Data JPA + Hibernate ORM, Spring Transaction Management, Liquibase

**Security**: Spring Security, HTTP Basic Authentication, BCrypt password encoding, Method-level Security, CORS Configuration, Session Management

**Web Layer**: Spring MVC REST Controllers, SpringDoc OpenAPI 3 + Swagger UI, Spring Multipart File Upload, Bean Validation

**Logging**: SLF4J + Logback

**Development Tools**: MapStruct, Lombok, Apache Commons IO

**Configuration**: Spring Configuration Properties, Spring Profiles

**Testing**: JUnit 5 + Spring Boot Test, Spring Security Test, H2 In-Memory Database, MockMvc
